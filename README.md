# Nannostomus (nannostomus)

Nannostomus provides a sex offender search API covering 771k+ offender records across all 50 U.S. states and 14 territories. It supports search by state and name, with monthly per-state record updates, deduplication, and 99.9% uptime backed by cloud infrastructure. A free tier covers up to 100 requests per month with tiered pay-as-you-go pricing beyond that.

### Nannostomus Sex Offender API
The Nannostomus Sex Offender API enables search of 771k+ offender records by state and name across all 50 U.S. states and 14 territories. Records are updated monthly with per-state change reporting and deduplicated within each state.
